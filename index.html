Chave da API

Para analisar os gabaritos, o app usa a IA da Anthropic. Você precisa de uma chave de API gratuita.

1. Acesse console.anthropic.com
2. Crie uma conta gratuita
3. Vá em "API Keys" e crie uma chave
4. Cole abaixo e salve

A chave fica salva só no seu celular (localStorage).
